# 🪴 서비스 소개

- 서비스명 : HelloHere
  - 서울에 위치한 사용자들이 위치 기반으로 교류 상대를 탐색하고, 실시간 채팅과 번역 기능을 통해 원활하게 소통할 수 있는 웹 앱 서비스
  - 미리 등록한 관심사를 통한 **점수 기반 AI 매칭**, **지도를 통한 직접 매칭**을 통해 매칭 서비스 구현
  - **실시간 채팅**, **상대 언어를 나의 언어로 번역** 하는 사용자 간의 원활한 소통을 위한 서비스 구현

![서비스 미리보기](./assets/preview.png)

# 🌱 기획 및 설계 포인트

- 글로벌 도시 서울의 시민과 관광객/외국인 주민들 간의 교류는 언어 장벽, 심리적 거리감, 문화 차이 등으로 자연스러운 대화가 시작되기 어렵습니다.
- 외국인 관광객의 경우 팬데믹 이후 불편 사례가 재증가 추세이며 **현장에서 즉시 도움**에 대해 어려움을 겪고 있습니다.
- 기존의 프로그램/플랫폼 등은 한정된 상담, 제한된 교류에 그치는 문제 배경을 바탕으로 기획되었습니다.
- **"지금, 이 장소에서 진짜 교류"** 라는 새로운 경험을 창출하여 외국인 주민 및 관광객에게는 특별하고 새로운 경험을, 시민에게는 글로벌 역량을 제공하고자 합니다.
- **HelloHere** 서비스는 실시간 매칭과 채팅을 기반으로 하며, 번역을 통해 언어 장벽을 해결합니다.

# 🌿 주요 기능

- **사용자의 언어 설정**  
  사용자의 선호 언어를 선택 받아 번역 기능을 제공합니다.
  영어, 한국어, 일본어, 중국어, 스페인어, 프랑스어 등을 선택할 수 있습니다.

- **사용자의 관심사 및 서비스 사용 목적 설정**  
  K-pop, K-Drama 등 다양한 사용자의 관심사와 서비스 사용 목적을 선택받습니다.
  설정된 관심사 및 목적은 주변의 교류할 상대를 찾을 때, AI가 해당 정보들을 통해 매칭 점수가 높은 상대를 추천합니다.

- **온보딩**  
  사용자가 처음 접속했을 때, 관심사/언어/목적 설정 과정을 안내하며 서비스의 핵심 기능을 소개합니다.
  사용할 수 있는 페이지와 기능 등에 대한 정보를 제공합니다.

- **실시간 매칭 서비스**
  - 저장된 사용자의 정보와 현재 사용자들의 위치 정보를 기반으로 AI가 추천 사용자를 매칭합니다.
  - 사용자는 현재 위치 기반으로 지도를 통해 주변 지역의 사용자를 탐색하고 직접 사용자의 정보를 조회, 교류를 신청할 수 있습니다.

- **채팅 및 실시간 번역 서비스**
  - 사용자는 매칭된 상대와 채팅을 통해 교류할 수 있습니다.
  - 채팅방에서 상대의 언어를 사용자가 선택한 선호 언어로 번역을 제공합니다.
  - 사용자 간의 직접 교류가 시작되면 단순 채팅이 아닌 STT(Speech-To-Text)를 활용하여 대화 번역을 제공합니다.

# 📚 기술 스택

### ✔️ Frontend

- **TypeScript**: 정적 타입으로 안정적인 개발 환경 구축
- **Next.js**: 서버사이드 렌더링(SSR) 및 라우팅 관리
- **MUI + TailwindCSS**: UI 컴포넌트 및 커스텀 스타일링
- **Kakao Map API**: 지도 기반 사용자 탐색 기능 구현
- **WebSocket API**: 브라우저 표준 WebSocket으로 실시간 매칭 및 채팅 처리

### ✔️ Infra & Tools

- **AWS Amplify**: 프론트엔드 배포
- **Notion**: 기획 문서 관리
- **Figma**: 와이어프레임 및 UI/UX 디자인
- **Git/GitHub**: 버전 관리 및 협업
